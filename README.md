> Fazendo um Tracker de acesso ao seu site

## Instalando

<p>Primeiro de tudo, ajuste a config.js</p>

```js
module.exports = {
  webhookId: "XXX",
  webhookToken: "XXX"
}
```

<p>Depois coloque estes comandos no console</p>

```js
npm install express discord.js --save
```

<p>E por fim no console</p>

```js
npm start
```
